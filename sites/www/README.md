# McpFile Documentation Site

Build with fumadocs, react, vite and cloudflare workers

## Getting Started

1. Install dependencies:

   ```bash
   npm install
   ```

2. Start the development server:

   ```bash
   npm run dev
   ```

3. Build for production:

   ```bash
   npm run build
   ```

4. Deploy to Cloudflare:
   ```bash
   npm run deploy
   ```
